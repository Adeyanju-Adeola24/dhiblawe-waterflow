HOW TO IMPORT TO GOOGLE SHEETS:
================================

1. Go to https://sheets.new (opens a blank sheet)
2. Click File → Import → Upload
3. Upload each CSV file one by one
4. When prompted, choose:
   - "Create new sheet" for each file
5. After importing all 8, rename the tabs to match the filenames:
   - users, clients, plate_numbers, trips, payments, deposits, settings
   - Delete the extra blank sheet (Sheet1)

ALTERNATIVE (simpler):
1. Go to https://sheets.new
2. Rename the first tab to "users"
3. Copy the header row from users.csv and paste into row 1
4. Click + (bottom left) to add a new tab
5. Repeat for each CSV file

Then share the sheet with your service account email.
